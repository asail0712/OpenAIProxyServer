🧠 OpenAI Realtime Proxy Server

這是一個 OpenAI 實時語音對話代理伺服器，
透過 WebSocket 與 gpt-4o-mini-realtime-preview 模型連線，
可讓客戶端（例如 Unity 或 Web 前端）以音訊進行即時對話。

🚀 專案特色

🔄 支援 WebSocket 雙向通訊

🎙️ 可傳送與接收 PCM16 音訊串流

🧩 使用 OpenAI Realtime API（模型：gpt-4o-mini-realtime-preview）

🗣️ 即時語音對話（Streaming Input / Output）

🧱 採用 .NET 8 Minimal API 架構

⚙️ 可作為中繼層（Proxy）提供安全 API Key 管理
